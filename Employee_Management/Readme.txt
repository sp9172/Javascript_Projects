# Employee Management System

A simple web-based Employee Management System built using HTML, CSS, Bootstrap, and JavaScript. This application allows users to manage employee records efficiently with the following core functionalities:

## Features

- **Add Employee:** Add new employee records with details like name, position, department, and contact information.
- **Edit Employee:** Update existing employee information directly from the list.
- **Delete Employee:** Remove employee records permanently from the system.
- **Search Employee:** Quickly search for employees by name, department, or other attributes.

## Technologies Used

- **HTML5** - For creating the structure of the web pages.
- **CSS3** - For styling the application.
- **Bootstrap 5** - For responsive design and pre-built UI components.
- **JavaScript (ES6)** - For handling user interactions and dynamic updates.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, etc.)

### How to Run

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd employee-management-system
   ```
3. Open `index.html` in your preferred browser.

## Usage

1. **Add Employee**
   - Click on the "Add Employee" button.
   - Fill in the required details.
   - Click "Save" to add the employee to the list.

2. **Edit Employee**
   - Click the "Edit" button next to the employee record.
   - Update the necessary details.
   - Click "Save" to apply changes.

3. **Delete Employee**
   - Click the "Delete" button next to the employee record.
   - Confirm the deletion to remove the employee from the list.

4. **Search Employee**
   - Use the search bar to filter employees by name, department, or role in real-time.
